# QDGZ300 服务器部署和压测指南

> 适用环境：麒麟 V10 ARM64 服务器  
> 默认源码目录：`/home/devuser/WorkSpace/qdgz300_backend`

## 1. 首次部署

### 1.1 拉取代码

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
git pull --ff-only origin codex/docs-reorg-cleanup
```

如果部署其他业务分支，请将分支名替换为目标分支。

### 1.2 构建

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
bash scripts/build/build_production.sh
```

脚本默认完成：

1. 环境检查
2. 共享依赖缓存准备
3. CMake 配置与构建
4. 单元测试

### 1.3 安装与启停

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
sudo bash deploy/install.sh build_production
sudo bash scripts/ops/enable_services.sh
sudo bash scripts/ops/start_services.sh
```

## 2. 验收与运维

### 2.1 服务检查

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
sudo bash scripts/ops/check_services.sh
```

### 2.2 日志查看

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
sudo bash scripts/ops/tail_logs.sh
```

## 3. 测试入口



## 4. 压测入口

### 4.1 快速验证

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
./build_production/tools/fpga_emulator --help
```

### 4.2 标准压测

```bash
cd /home/devuser/WorkSpace/qdgz300_backend
chmod +x tools/run_receiver_standard_bench.sh
./tools/run_receiver_standard_bench.sh --build-dir build_production --quick
```
