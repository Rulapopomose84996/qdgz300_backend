# 依赖与工具链清单

## 1. 核心语言与构建基础

- C++17
- CMake 3.16.x
- Ninja
- ARM64 / aarch64

## 2. 主要第三方库

| 组件 | 主要用途 |
|---|---|
| `yaml-cpp` | YAML 配置解析 |
| `spdlog` | 日志输出 |
| `fmt` | 字符串格式化 |
| `nlohmann_json` | JSON 序列化 |
| `Google Test` | 单元测试 |
| `Google Benchmark` | 基准测试 |

## 3. GPU 与平台相关依赖

| 组件 | 主要用途 |
|---|---|
| CoreX SDK 4.3.8 | GPU 编译器与运行时 |
| clang 18.1.8 | 服务器 GPU 路径编译器 |
| `libnuma` | NUMA 感知分配与绑定 |
| `pthread` | 线程与亲和性管理 |

## 4. 系统接口

- `recvmmsg` / `sendmmsg`
- `epoll`
- `SO_REUSEPORT` / `SO_RCVBUF` / `SO_BUSY_POLL`
- `numactl` / `taskset`
- `systemctl` / `journalctl`

## 5. 当前工具链主线

### 5.1 Linux 服务器

- 默认脚本：`scripts/build/build_production.sh`
- 默认构建目录：`build_production`
- 默认依赖来源：共享离线包目录 + 共享原生缓存目录

### 5.2 WSL

- 默认脚本：`scripts/build/build_wsl_cross.sh`
- 默认构建目录：`build_wsl_cross_dev`
- 默认 GPU：关闭

## 6. 说明

- 具体路径和环境变量以 [部署基线](../../基线/麒麟V10_ARM64_CoreX_CUDA_部署基线.md) 为准。
- 依赖清单用于说明工程组成，不代替构建脚本和基线文档。
